ICT Basics: I/O Systems

1. Input

Getting data INTO the computer.

  • Manual: Keyboard, Mouse.
  • Direct: Barcode (Shops), QR Code, Sensor (Auto-lights).

2. Output

Getting info OUT to the user.

  • Soft Copy: Monitor, Speaker.
  • Hard Copy: Printer (Laser/Inkjet).

Input Devices

Device Description / Key Point
QR Code 2D barcode that holds more data than standard barcodes. Used for URLs and payments.
Touchpad Tactile sensor that translates finger movement to cursor movement. Standard on laptops.
Smart Card Card with an embedded chip (e.g., Octopus card). Uses RFID for contactless data transfer.
Biometric Measures unique physical traits (Fingerprint, FaceID). High security; impossible to "lose" password.

Display Specifications

  • Screen Size: Measured diagonally in inches.
  • Resolution: Total number of pixels (Width x Height). Higher = sharper image.
  • Pixel Density (PPI): Pixels per inch. Higher PPI makes individual pixels invisible to the eye.
  • Colour Depth: Number of bits used to represent a color. E.g., 24-bit depth allows for millions of colors.

Connector Standards

Port Signal Type DSE Fact
VGA Analog Old standard; blue plug; no audio.
HDMI Digital Transmits both High-Def video and audio in one cable.
DisplayPort Digital Similar to HDMI but often used for high-end PCs/monitors.
DVI Digital/Analog Video only; usually found on older desktop monitors.

Other Output Devices

  • Monitor: Primary output for visual data (Soft copy).
  • Speakers / Headphones: Converts digital signals into sound.
  • Projector: Displays images onto a large flat surface; used for presentations.
  • Plotter: A specialized "printer" that uses pens to draw high-precision architectural blueprints.
  • Actuators: Mechanical parts that move (e.g., a motor that opens a smart gate).
