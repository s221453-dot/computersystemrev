Computer Hardware & CPU Architecture

The System Unit & Motherboard

System Unit: The main case that houses the internal components.

Motherboard: The main circuit board. It acts as the "highway," connecting the CPU, RAM, and all other peripherals via buses.

1. The Central Processing Unit (CPU)

The "Brain" of the computer. Its performance is measured by:

  • Clock Speed: Measured in Hertz (Hz). The number of cycles performed per second.
  • Clock Cycle: The smallest unit of time for a processor to perform an instruction.
  • Cores: Multiple processors on one chip. More cores = better multi-tasking.
  • Word Length: The number of bits a CPU can process at once (e.g., 64-bit).

Internal Components of CPU

Component Function
Control Unit (CU) Manages the execution of instructions and controls the flow of data.
ALU Arithmetic Logic Unit. Performs math ($+, -, \times, \div$) and logic (AND, OR, NOT).
Registers High-speed, tiny storage locations inside the CPU.
Cache Memory Very fast memory between CPU and RAM. Stores frequently used data to speed up processing.

Special Purpose Registers (The "Big Four")

  • Program Counter (PC): Holds the address of the next instruction.
  • MAR: Memory Address Register. Holds the address of the current data being read/written.
  • MDR: Memory Data Register. Holds the actual data fetched from or waiting to be sent to memory.
  • Accumulator: Stores the intermediate results of calculations from the ALU.

2. The Machine Cycle

The process of handling one instruction: Fetch → Decode → Execute → Store.

3. Memory Hierarchy

Main Memory: RAM vs. ROM

Feature RAM (Random Access Memory) ROM (Read Only Memory)
Volatility Volatile (Data lost when power is off) Non-Volatile (Data stays)
Usage Current running programs/data. Startup instructions (BIOS/Firmware).

Secondary Storage

Used for permanent storage. Categorized by Access Method:

  • Sequential Access: Must read data in order (e.g., Magnetic Tape). Slow for finding specific files.
  • Direct Access: Can jump straight to any file (e.g., Hard Disk, SSD). Fast.
